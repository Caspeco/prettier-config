# prettier-config

Shared config for prettier

## Usage

Add the dependency and mapping for prettier configuration:

```
// package.json
{
  "name": "my-project",
  "version": "1.0.0",
  "prettier": "@caspeco/prettier-config",
  "devDependencies": {
    "@caspeco/prettier-config": "github:caspeco/prettier-config#1.0.0",
  }
}
```

Run `npm install`.
